# **App Name**: HN TextileVerse

## Core Features:

- Multilingual Support: Provide content in Turkish, Arabic, and English to cater to the primary target audiences.
- Product Catalog with Filtering: Enable users to explore fashion items by season, type, models, collections, and product.
- WhatsApp Integration: Implement a direct WhatsApp chat feature to streamline inquiries and order processing.
- Admin Panel for Content Management: Create an admin interface for managing images, product details, inventory, and contact information.
- Admin Image Uploader: Allow uploading of image links for display in product listings in webp format
- Inventory Status: Update products on the website with a designation of either available, or sold out
- Mobile App Conversion: Enable functionality of the site via an Android app, converted from the Nextjs source code
- Video URL: Optional field for pasting a video link (YouTube / YouTube Shorts / Vimeo). A small 'Watch Video' button or play icon is placed over the main product image. Clicking it will pop up the video, showcasing the model moving with the clothes. This maintains a clean and uncluttered design.

## Style Guidelines:

- Background color: Light gray (#F0F0F0) provides a clean and modern canvas for showcasing fashion designs.
- Primary color: Muted blue (#6A869A) reflects trust and sophistication.
- Accent color: Soft white (#FFFFFF) is used to give the website the feeling of refinement.
- Headline font: 'Playfair' (serif) evokes an elegant and fashionable tone; body text: 'PT Sans' (sans-serif) is easily readable.
- Employ a clean, minimalist layout with clear navigation, emphasizing visual content and ease of use.
- Use simple, elegant icons for categories and functions, ensuring they are intuitive and blend seamlessly with the design.
- Implement subtle animations for image transitions and user interactions, enhancing the overall user experience without being distracting.
